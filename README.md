# Nomflix
-----

이 프로젝트는 영화나 TV프로그램에 대한 정보를 공유 하는 기능을 목표로 두고있다.
클론 코딩 프로젝트를 통해 React에 대한 주요 정보들을 이해와 학습하는데 중점을 두고있다.
